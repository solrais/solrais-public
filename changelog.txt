# 🚀 Release v1.4.3 — Relay Expansion & AI Refinements

> This update adds Relay Node 3, the `/fix_missing_rank` command, verification auto-reply in the support channel, significant AI improvements, and various bug fixes across the board.

### 🤖 AI Chat Improvements

- **Tool Enforcement Overhaul**: Fixed a critical bug where the AI was answering from its own training knowledge instead of calling tools — it now properly looks up data before every response
- **Image Generation**: Added a new callable tool that lets the AI generate images via the Magnific API
- **Vision Support**: The AI can now analyze images sent in chat using Groq's vision capabilities
- **Creator Knowledge**: Solrais now knows who built it — ask about @dxrkfire or @n3ru__ and it can tell you
- **2000-Character Splitting**: Long AI responses are no longer truncated — they're split across multiple messages automatically
- **Personality Toning**: Reduced the excessive Sol's RNG obsession in default responses; personality is now more balanced
- **Safeguard Pipeline**: New backend pipeline with TF-IDF knowledge retrieval, HHEM hallucination detection, and Groq-based content filtering

> Solrais AI is continuously improving. Report quirks or weird responses to staff.

### 🆕 New Commands & Systems

- **`/fix_missing_rank`**: Replaces the old `/fix_afk_role`. Instead of just removing AFK roles, this scans all members and restores the macroer rank to anyone who lost it due to previous bugs. Shows per-server stats on fixed vs. already-had-rank counts.
- **Verification Auto-Reply**: The unverified support channel now has a smart auto-responder. If you ask about verification, it posts step-by-step instructions. If you say the commands aren't working, it defers to staff automatically.
- **Relay Node 3**: Cyberspace server now has its own dedicated relay node, fully integrated with the watchdog handler

> `/fix_afk_role` has been **removed**. Use `/fix_missing_rank` instead.

### 🖥️ Multi-Server & Configuration

- **Server 3 Config Fix**: Server 3 was incorrectly using Server 2's category IDs — channels, roles, AFK, and dormant systems now work correctly on Cyberspace
- **solraisconfig Menu Fix**: Resolved a case-sensitive custom ID bug that prevented the config menu from loading properly
- **Config UI Modal Fix**: Server 3's modal was extracting the wrong server ID — settings now save to the correct server
- **Staff Role IDs**: Updated for proper multi-server support across all three servers

### 🌐 Relay & Biome Systems

- **Free Biome Webhook Added**: Server 1 was missing the `free` webhook entirely — relay bots now correctly route Windy, Rainy, and Snowy through the dedicated free webhook
- **Weather Relay Simplified**: Removed the separate weather relay system; all free biomes now use the unified free webhook channel
- **Merchant & Aura Detection**: Improved Mari, Jester, and Rin detection with better pattern matching

### 🛡️ Bug Fixes & Improvements

- **`/force_roblox_link` Permission Fix**: No longer requires full Administrator permission — checks for staff role IDs at runtime instead
- **Duplicate Embeds Cleaned**: No more duplicate webhook processing or "No Macro Selected" ghost embeds
- **Ping Pings Eliminated**: Fixed an issue causing unintended ping mentions in relay messages
- **Macro Time Leaderboard Accuracy**: Session duration calculations improved for more accurate rankings
- **Watchdog Alert Cleanup**: Alerts now auto-clear when a recovering node comes back online instead of persisting
- **Wrong Ping Role Fix**: Watchdog alerts no longer ping the incorrect role
- **User Storage Increased**: Folder file limit raised from 25 to 50 for better scalability
- **`/stats` Path Display**: The stats command now shows the user's data file path for debugging
